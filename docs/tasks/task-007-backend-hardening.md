- Priority: `medium`

## Goal

운영 가능한 수준으로 백엔드 품질과 보안을 보강한다.

## Scope

- DB 마이그레이션 체계 도입 여부 결정
- seed 스크립트 정리
- 환경 변수 검증 강화
- cookie / secret / token 만료 정책 고정
- 에러 응답 포맷 정리
- OpenAPI 태그/설명/예시 보강
- 통합 테스트 확대

## Done When

- 로컬/개발 환경 재현성이 높아짐
- 보안 관련 기본 설정이 문서와 코드에서 일치함
- Swagger 문서만으로 주요 API 확인이 가능함
