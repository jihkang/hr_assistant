# Task 003: Assets and Rentals

- Status: `todo`
- Priority: `high`

## Goal

총무 도메인의 핵심 기능인 자산 조회와 대여 워크플로우를 구현한다.

## Scope

- `assets` ORM 모델 구현
- `asset_rentals` ORM 모델 구현
- 자산 목록 조회 API
- 자산 상세 조회 API
- 대여 가능 자산 검색 API
- 대여 신청 API
- 관리자 승인/반려/반납 API

## Done When

- 일반 사용자가 대여 가능 자산을 검색할 수 있음
- 사용자가 자산 대여 신청 가능
- 관리자가 승인/반려/반납 처리 가능
- 권한 및 상태 전이 테스트 포함
